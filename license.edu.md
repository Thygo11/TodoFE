Educational Use Policy

This repository is a fork of an existing project and has been modified with
additional tests for the purpose of teaching and learning unit testing.

Policy: The educational modifications in this repository are intended solely
for instructional use in classrooms, workshops, and self-study. They are
provided to help students practice testing techniques and understand test
coverage.

Original license terms for the upstream project still apply to all original
source code and materials. See [license.md](license.md).
